# Language Translation Tool

A web-based language translation application built with Python and Streamlit that allows users to translate text between multiple languages using Google Translate API.

## Features

- Translate text between 14 different languages
- Clean and simple web interface
- Real time translation results
- Warning messages for empty input and same language selection
- Copy translated text easily

## Technologies Used

- Python 3.x
- Streamlit
- Deep Translator (Google Translate API)

## Installation

1. Clone the repository
2. Navigate to project directory
3. Install required libraries
